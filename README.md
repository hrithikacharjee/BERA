BERA is a feature-enriched, transformer-based, multi-task architecture for
joint sentiment and emotion analysis of Bangladeshi e-commerce product reviews.

## Key Characteristics
- Custom SentencePiece tokenizer for Bangla + English + Banglish + Emojis
- Feature-enriched transformer encoder
- Multi-task learning (Sentiment + Emotion)
- Drive-persistent training (Colab account switch safe)
- Java-ready analytics export (word count, emoji count, CMI, metrics)

## Repository Usage
- Python: model training, feature extraction, evaluation
- Java: visualization dashboard (future)

This repository is version-controlled for academic supervision and reproducibility.
